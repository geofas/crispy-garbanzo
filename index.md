# Harness Engineering Trainer

A self-directed curriculum for becoming a subject-matter expert in
**harness engineering** — the discipline of designing, building, and
operating the scaffolding that turns a language model into a capable
agent.

## Start here

1. Read [Module 00 — What is harness engineering?](curriculum/00-what-is-harness-engineering.html) first.
2. Work through the modules in order; each exercise set lives under
   [Exercises](exercises/).
3. Build the capstone in [Module 10](curriculum/10-capstone.html).
4. Keep the [Glossary](glossary.html) and [References](references.html)
   open as you go.

The full repo (including the capstone starter code) is on
[GitHub](https://github.com/geofas/crispy-garbanzo).
